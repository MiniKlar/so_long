**So_Long** - 2D Game Development

**What it is:** A simple 2D game built from scratch using the MiniLibX graphics library, featuring player movement, collectibles, and win conditions.

**Business Value:**
- **User Experience:** Understanding of interactive application development
- **Graphics Programming:** Foundation for multimedia and gaming industries
- **Project Management:** Complete software development lifecycle from concept to completion

**Technical Skills Developed:**
- Graphics programming and rendering
- Event handling and user input processing
- Game logic and state management
- File parsing and map validation
